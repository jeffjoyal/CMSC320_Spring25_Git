
Q1: Why might we consider using Git?

Answer: It allows for easy collaboration with a large group of people, as well as providing a great framework

Q2: What field could data science be applied to? Give an Example.

Answer: The Machine Learning Industry has huge use for data science, because a lot of the field involves
		feeding huge amounts of data to a machine learning model in order for it to learn patterns
		from that data.
